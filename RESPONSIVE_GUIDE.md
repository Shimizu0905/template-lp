# 1920px カンプのレスポンシブ対応ガイド

## 📐 現在の設定

- **デザインカンプ**: 1920px
- **inner 幅**: 1200px
- **ベースフォントサイズ**: 16px

## 🎯 関数の使い分け

### 1. `prem()` - 固定サイズ（rem 変換のみ）

カンプの px 値をそのまま rem に変換します。画面サイズに応じたスケーリングは**行いません**。

```scss
// 例: カンプで100px → 6.25rem（16px基準）
width: prem(100); // 6.25rem
```

**使用場面:**

- フォントサイズ（テキストは固定サイズが適切な場合）
- 最小/最大サイズを指定したい要素
- モバイルで固定サイズにしたい要素

---

### 2. `prem-vw()` - 1920px 基準の vw 計算（推奨）

1920px のカンプ値をそのまま入れると、画面幅に応じて自動的にスケーリングします。

```scss
// 例: カンプで100px → 1920px画面で100px、960px画面で50px
width: prem-vw(100); // 5.208vw（1920px基準）
```

**使用場面:**

- レイアウト幅、余白、コンテナサイズ
- 画像サイズ
- 画面幅に応じてスケールさせたい要素

**注意:** 小さな画面で小さくなりすぎる可能性があるため、`clamp()`と組み合わせることを推奨します。

---

### 3. `prem-clamp()` - 最小・最大値を指定したスムーズなスケーリング（最推奨）

最小値と最大値を指定して、スムーズにスケーリングします。

```scss
// 例: 最小20px、最大100px、1920px基準でスケーリング
width: prem-clamp(20, 100, 1920);
// → clamp(1.25rem, 5.208vw, 6.25rem)
```

**使用場面:**

- レスポンシブ対応が必要なすべての要素
- 小さすぎたり大きすぎたりしないようにしたい要素

---

## 📝 実装例

### 例 1: コンテナ幅

```scss
.container {
  // カンプで1200px → 最小800px、最大1200pxでスケーリング
  max-width: prem-clamp(800, 1200, 1920);
  margin: 0 auto;
  padding: 0 prem-clamp(20, 40, 1920);
}
```

### 例 2: フォントサイズ

```scss
.title {
  // カンプで48px → 最小24px、最大48pxでスケーリング
  font-size: prem-clamp(24, 48, 1920);
  line-height: 1.5;
}
```

### 例 3: 余白

```scss
.section {
  // カンプで80px → 最小40px、最大80pxでスケーリング
  padding: prem-clamp(40, 80, 1920) 0;
}
```

### 例 4: 固定サイズが必要な場合

```scss
.icon {
  // カンプで24px → 常に1.5rem（24px）
  width: prem(24);
  height: prem(24);
}
```

---

## ⚠️ 注意点

1. **`prem()`だけでは完全なレスポンシブにはなりません**

   - rem は親要素の font-size に依存します
   - 画面幅に応じたスケーリングは行いません

2. **`prem-vw()`は小さな画面で小さくなりすぎる可能性があります**

   - 必ず`clamp()`や`prem-clamp()`と組み合わせることを推奨

3. **`prem-clamp()`を積極的に使いましょう**
   - 最小値と最大値を指定することで、どの画面サイズでも適切な表示になります

## 🔧 カスタマイズ

### デザインカンプの基準幅を変更する場合

`src/scss/foundation/variables.scss`で変更：

```scss
$design-width: 1920px; // 変更したい幅に
```

`prem-vw()`と`prem-clamp()`の第 3 引数も変更：

```scss
width: prem-clamp(20, 100, 1440); // 1440pxカンプの場合
```
